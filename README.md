# esx_legalfarming

I changed ESX_Drugs and added Legal Farming stuff. 

## Installation
- Import `esx_legalfarming_de.sql` or `esx_legalfarming_en.sql` in your database
- Add this in your `server.cfg`:

```
start esx_legalfarming
```
